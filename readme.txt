A plugin that will automatically attach a featured image.  It will randomly pick from your
media any file that is NOT attached and starts with a EW- (Use As Feature).
== Description ==

Well here is a plugin that will randomly pick an image that you upload through your back office media interface.
You just have to make sure the file name begins with a [EW-] (for Use Feature) and it is unattached. Just upload it and don’t use it for a post.

== Installation ==

Log into the administration area of your blog and click on Plugins -> Add New in the menu. Search for the name of this plugin and then click install on the right side, and then again in the pop-up window.
